## Contributing

**Working from a fork?** See our [Fork Workflow Guide](docs/DEVELOPMENT.md#fork-workflow).

### Quick Start for Collaborators

1. **Fork this repo** to your GitHub account
2. **Clone your fork** locally
3. **Run tests** before making changes:
   ```bash
   npm install
   npm test  # Must pass before submitting
   ```
4. **Make your changes** in a feature branch
5. **Submit a PR** from your fork to this repo's `main` branch
6. **Wait for review** - Zarika will review and merge
